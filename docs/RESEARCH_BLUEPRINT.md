# Elite v1 — Best-in-Class Blueprint (Kuwait, 2026)

Research-derived, prioritized backlog to beat Talabat / Keeta / Noon / Amazon.ae / AliExpress.
**The winning wedge:** electronics + **professional installation** + warranty/service in one tracked flow — the one thing no incumbent owns in Kuwait. Win parity on P0, then over-invest in the installation/field-service moat (P1), with AI as the multiplier (P2).

## P0 — table stakes (parity; lose sales/trust without them)
| Feature | Why | Effort |
|---|---|---|
| KNET (primary) + cards + Apple/Google Pay via Tap/MyFatoorah | ~80% of Kuwait online payments are KNET | M |
| Guest checkout + saved KW address (governorate/area/block/street) + map pin | Friction is the #1 GCC mobile conversion killer | M |
| Real-time GPS tracking + dynamic ETA (±2 min) + driver profile | Keeta/Talabat set the expectation | L |
| WhatsApp order updates + COD confirmation flow | Cuts COD cancellations ~15–25% → 3–7%; #1 ROI channel | M |
| Arabic-first RTL + correct hreflang (ar-KW / en-KW) | Wrong language served kills SEO + UX | M |
| Core Web Vitals pass (LCP<2.0s, INP<200ms, CLS<0.1) + Arabic font subsetting | ~24% higher conversion; ranking signal; Arabic sites fail INP | M–L |
| Clear returns + warranty (≥1yr) surfaced at PDP/checkout | Trust vs AliExpress grey-market fear | S |
| Verified-purchase reviews with photo/video | Social proof; competitors under-invest | M |

## P2 — advanced edge (once core is live)
AI recommendations (cross-sell/upsell), AI support copilot (AR/EN), AI multi-stop driver routing, demand forecasting (stock + installer capacity), WhatsApp referral program, visual search, back-in-stock/price-drop alerts, digital warranty card → one-tap service booking.

## Kuwait-specific notes
- **Payments:** KNET must be default/top, not buried. Apple Pay high-impact (iPhone penetration). Keep COD (~10%) with WhatsApp confirmation. BNPL expected on electronics.
- **WhatsApp** is the highest-ROI channel in GCC — ordering, updates, COD confirm, cart recovery, support, referrals.
- **SEO:** Arabic-first (not translated), RTL hygiene, structured data + Merchant/Meta feeds, subset Arabic fonts for INP.
- Cross-GCC stats (UAE/KSA) are directional — validate against the live funnel.
